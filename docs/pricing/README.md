## How to Use These Files

1. **For an AI prompt:** Upload the relevant business type file + `FEATURES_UNIVERSAL.md` together.
2. **For a website pricing page:** Each file maps to one business-type tab/section.
3. **For a sales deck:** Use the "Recommended Plan" section at the bottom of each file.
4. **Source of truth:** All features are derived from `Dukan_x/lib/core/billing/business_type_config.dart` and `Dukan_x/lib/guards/plan_tier_guard.dart`.
